# AZ Clean Auto Chunk Sorter

This is an iPad-friendly offline web app.

## What it does
- Loads pasted text or a .txt file
- Converts to uppercase
- Removes symbols/numbers/punctuation
- Keeps lines with 2+ words
- Removes exact duplicates
- Sorts into A-Z buckets
- Exports MASTER_AZ_INDEX.txt
- Exports selected letter file
- Exports REVIEW_NOT_DELETED.txt

## iPad install
1. Upload these files to GitHub Pages, or open index.html locally.
2. Open the page in Safari.
3. Tap Share.
4. Tap Add to Home Screen.
5. Open AZ Clean from your iPad Home Screen.
